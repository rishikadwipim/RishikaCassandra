# RishikaCassandra

## Description

RishikaCassandra is a project that demonstrates how to use Apache Cassandra for handling big data. The project showcases the integration of Cassandra with various big data tools, helping manage large-scale datasets with a distributed database system. It includes various features such as data modeling, query execution, and visualizations using Cassandra's capabilities.

## Features

- Integration with Apache Cassandra
- Query execution and data retrieval using Cassandra
- Secure connection to Cassandra using `secure-connect-rishikadwipi.zip`
